Ice Cubes is a free, open-source application for accessing Mastodon, the decentralized social network. Built entirely in SwiftUI, it runs natively on iOS, iPadOS, macOS, and visionOS. You can connect to any Mastodon instance, browse your timeline, interact with other users, and post updates and media — all from a fast, lightweight native app that feels at home on every Apple platform.

## Platform support

Ice Cubes runs on all modern Apple platforms from a single codebase:

| Platform | Notes |
|----------|-------|
| **iOS** | Full feature set on iPhone |
| **iPadOS** | Dedicated sidebar UI for larger screens |
| **macOS** | Native Mac app with sidebar navigation |
| **visionOS** | Runs on Apple Vision Pro |

The app uses a dedicated sidebar layout on macOS and iPadOS, making the most of the available screen real estate on those platforms.

## Architecture

Ice Cubes is organized into independent Swift packages under `/Packages/`, each responsible for a specific slice of the application — network communication, data models, UI components, theming, and more. This modular structure makes the codebase easier to navigate, maintain, and contribute to.

The app follows a modern SwiftUI architecture: views are pure state expressions using SwiftUI's native property wrappers (`@State`, `@Binding`, `@Observable`, `@Environment`), with no ViewModels for new code. All asynchronous work uses Swift's `async/await` concurrency model.

<Note>
  Ice Cubes is a great starting point for learning SwiftUI. The codebase covers building layouts, working with data, handling user interaction, and real-world patterns across a production app.
</Note>
